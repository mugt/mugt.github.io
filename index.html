Get Started
git init project_dir Create repository in project_dir
git clone git@github.com:palmin/mug.git Clone repo to local machine
git checkout -b feature
Create and shift to branch feature
git add src
Stage files in src dir
git reset src/main.c
Remove file main.c in src dir from staging
git commit -m "new sources"
Commit staged snapshot with message
git pull origin
Fetch from origin remote and merge with local
git push origin feature
Push commits on feature branch to origin
History & changes
Git status
Show repo status and changed files
git merge
Merge current branch back to master branch
git log
Display list of latest commits
git diff
Show difference made since last commit
git reflog
Information about important changes
git branch -a
List all branches in your repo
git help
Get help
