Get Started
git init my_project_dir Create repository in my_project_dir
git clone git@github.com:palmin/mug.git Clone repo to local machine
git checkout -b coolstuff
Create and shift to branch coolstuff
git add src
Stage files in src dir
git reset src/main.c
Remove file main.c in src dir from staging
git commit -m "New sources"
Commit staged snapshot with message
git pull origin
Fetch from origin remote and merge with local
git push origin coolstuff
Push commits on coolstuff branch to origin
History & changes
Git status
List files files are staged
git merge
Merge current branch back to master branch
git log
Display the intere commit history
git diff
Show unstaged changes
git reflog
Show changes to the local repos HEAD
git branch -a
List all branches in your repo
git help
Get help
